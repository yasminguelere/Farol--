# Farol-
Esse código serve para controlar um semáforo (farol de trânsito) inteligente usando uma placa de circuito, onde você tem três LEDs fazendo o papel das luzes e três botões para mudar o ritmo delas. O grande objetivo desse projeto é permitir que você altere o comportamento do farol na hora que quiser, apenas escolhendo qual botão apertar.

Se você apertar o primeiro botão, o farol entra no modo normal de rua, ligando uma luz por 3 segundos, depois a outra por 5 segundos e a última por 1 segundo e meio, repetindo o ciclo. Se você apertar o segundo botão, ele entra em modo de alerta, piscando cada luz bem rápido, uma atrás da outra, a cada meio segundo. E se você apertar o terceiro botão, ele liga todas as luzes juntas por 2 segundos e desliga por 1 segundo, como se fosse um sinal de emergência.

Esse projeto serve basicamente para criar o sistema de trânsito de uma maquete ou testar lógicas de automação de semáforos para uma cidade.
